Usage: ls[OBJ] [PATH]   
Recursively displays specified object type from given path.

Where OBJ can be:   
ten   
site   
bldg   
room   
rack   
dev   
ac   
aisle   
tile   
cabinet   
corridor   
panel   
separator   
sensor       

EXAMPLE   

    lsten   
    lsten ../../Physical/TenantB
    lscabinet
    lspanel
