Usage: ls[OBJ] [PATH]   
Recursively displays specified object type from given path.

Where OBJ can be:      
site   
bldg   
room   
rack   
dev   
ac   
aisle   
tile   
cabinet   
corridor   
panel   
separator   
sensor       

EXAMPLE   

    lssite   
    lssite ../../Physical/SiteB
    lscabinet
    lspanel
